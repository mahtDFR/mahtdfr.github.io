chris says:

i'm using github pages
it's free
basically you can put an index.html in a repo, tell github that it's a website and voila: username.github.io is now your site
then you add a custom domain to it using cname records
just get a domain from any decent registrar that also supports DNS configuration (to enter the cname record for forwarding to github)
I got mine from antagonist (dutch registrar/hosting company that started near where I studied)
we also use github pages to publish our design system components: https://ultimaker.github.io/stardust-web/
github offers some decent automation pipelines (github actions) to run a build job to turn your dynamic code into a static website, which you can then publish like this
